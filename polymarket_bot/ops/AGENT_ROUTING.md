# Agent Routing

## Patch Agent
- Scope: bot_v5.py, strategy/risk logic changes, launch scripts
- Output: patch commit + changelog entry

## Analytics Agent
- Scope: KPI snapshots, daily reports, version comparisons
- Output: markdown reports + csv/json summaries

## Runtime Monitor Agent
- Scope: stale state, duplicate process checks, restart advisories
- Output: watchdog alerts + health status
